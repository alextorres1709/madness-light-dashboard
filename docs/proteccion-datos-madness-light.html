Protección de Datos y Privacidad

Documento para Madness Light — Cómo tratamos y protegemos la información

NodexAI no vende, comparte ni cede datos a terceros bajo ningún concepto. Toda la información de vuestros clientes es estrictamente privada, de vuestra propiedad, y se utiliza exclusivamente para el funcionamiento de la plataforma contratada.

1. Principio fundamental

La relación entre NodexAI y Madness Light se basa en un principio claro:

Los datos son vuestros. Nosotros solo los custodiamos.
NodexAI actúa como encargado del tratamiento de los datos — los almacenamos y procesamos siguiendo vuestras instrucciones, pero no tenemos ningún derecho sobre ellos. En caso de baja del servicio, se entrega una copia completa y se eliminan de nuestros servidores.

4. Dónde se almacenan los datos

Componente Ubicación Proveedor Seguridad
Base de datos principal Servidores en la UE Railway (PostgreSQL) Cifrado en reposo y en tránsito (SSL/TLS)
Copia local Equipo del administrador SQLite (dentro de la app) Cifrado del disco del equipo
Backups Servidores en la UE Railway (automáticos) Copias diarias con retención de 7 días
API de WhatsApp Servidores de Meta Meta (WhatsApp Business API) Cifrado end-to-end (protocolo Signal)
IA (ChatGPT) Servidores de OpenAI OpenAI Los mensajes NO se usan para entrenar modelos (API de pago)
Todos los proveedores utilizados cumplen con el RGPD (Reglamento General de Protección de Datos) y tienen sus servidores principales en la Unión Europea o cuentan con las cláusulas contractuales tipo (SCCs) para transferencias internacionales.

5. Medidas de seguridad

Técnicas

  • Cifrado SSL/TLS en todas las comunicaciones entre la app, el servidor y las APIs
  • Contraseñas hasheadas — las contraseñas de los usuarios del panel se almacenan con hash (bcrypt), incluso nosotros no podemos verlas
  • Acceso restringido — solo el administrador de Madness Light puede acceder al panel completo. Los empleados pueden tener permisos limitados
  • Tokens de API rotativos — las claves de acceso a WhatsApp y OpenAI se rotan periódicamente
  • Backups automáticos — copias de seguridad diarias de la base de datos con retención de 7 días

Organizativas

  • Acceso mínimo necesario — solo el personal técnico de NodexAI accede a los servidores, y exclusivamente para mantenimiento
  • Sin acceso a contenido — el equipo técnico de NodexAI no lee las conversaciones ni los datos personales de los clientes salvo que Madness Light lo solicite expresamente para resolver una incidencia
  • Confidencialidad — todo el equipo de NodexAI está sujeto a obligaciones de confidencialidad

6. Servicios de terceros utilizados

La plataforma utiliza los siguientes servicios externos. Ninguno de ellos recibe, almacena ni tiene acceso a la base de datos de clientes de Madness Light:

Servicio Para qué se usa ¿Recibe datos de clientes? Cumple RGPD
WhatsApp Business API (Meta) Envío/recepción de mensajes Solo el número de teléfono y el contenido del mensaje (cifrado E2E) Sí
OpenAI (ChatGPT) Respuestas del bot IA El texto de la conversación (sin nombre ni teléfono). No se usa para entrenar modelos. Sí
Railway Hosting del servidor y base de datos Almacena la BBDD (como encargado del tratamiento) Sí
n8n (self-hosted) Automatizaciones (cumpleaños, campañas) Se ejecuta en nuestros servidores, no envía datos fuera Sí

7. Derechos del cliente final

Los clientes de Madness Light (las personas cuyos datos se almacenan) tienen los siguientes derechos según el RGPD:

Derecho Qué significa Cómo se ejerce
Acceso Saber qué datos suyos tenemos almacenados Solicitud a Madness Light → exportamos los datos del cliente desde el panel
Rectificación Corregir datos incorrectos Se editan directamente desde el panel
Supresión Eliminar todos sus datos (derecho al olvido) Se eliminan del panel y de la base de datos. Irreversible.
Oposición Dejar de recibir mensajes Se marca como opt-out en el panel. No recibirá más campañas ni cumpleaños.
Portabilidad Recibir sus datos en formato legible Exportación CSV desde el panel
Responsabilidad: Madness Light es el responsable del tratamiento de los datos de sus clientes. NodexAI es el encargado del tratamiento. Madness Light decide qué datos se recogen y para qué se usan; NodexAI los almacena y procesa siguiendo sus instrucciones.

8. ¿Qué pasa si Madness Light deja de usar NodexAI?

  • Exportación completa — Se entrega toda la base de datos en formato CSV (clientes, conversaciones, eventos, estadísticas)
  • Eliminación total — En un plazo máximo de 30 días tras la baja, se eliminan todos los datos de nuestros servidores y copias de seguridad
  • Certificado de destrucción — Si se solicita, emitimos un certificado confirmando que los datos han sido eliminados
  • Sin permanencia — No retenemos datos como "penalización" ni por ningún otro motivo
